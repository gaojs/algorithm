小明要从大门到卧室，中间要穿过一个M列N行的房间矩阵。
大门（顶部一整行）
-----------
0 7 9 6 0 0
0 0 1 0 0 0
2 0 3 0 4 0
0 0 5 0 0 0
0 0 8 0 0 0
-------------------
卧室（底部一整行）
房间逐个亮灯，亮灯顺序，会给出的。
小明怕黑，必须等房间亮灯后，才能进入。
小明只能进入四周相邻的房间。
求：至少亮几盏灯，小明才能从大门穿过房间矩阵，到达卧室。

测试用例1（如上图）：
输入：
6 5
3 2
1 3
3 3
5 3
3 4
4 1
2 1
3 5
3 1
4 3
1 4
1 1
-1
输出：
9

说明：
首行是M=6列，N=5行。
其次是X=3列，Y=2行亮灯。
直到输入-1时，结束亮灯顺序。
如上图，非0数字表示亮灯的顺序号。
